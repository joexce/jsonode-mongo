# mongo-using-node
Cara kerja aplikasi ini sangat mudah , nodejs mengambil data dari file.json lalu di teruskan ke mongo.<br>
> it will get data from file.json and proceed to mongo database

### Preparation
```
cd mongo-using-node
npm install mongodb --save
```

### Run Application
```
nodejs app.js
```
